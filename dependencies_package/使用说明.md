# 短视频生成器 - 离线安装包使用说明

## 📦 这是什么？

这是一个**离线依赖包**，包含了短视频生成器运行所需的所有Python库。

客户使用这个包可以**跳过在线安装步骤**，直接运行程序。

---

## 🚀 如何使用（3步完成）

### 第一步：下载并解压
1. 下载 `dependencies_package.zip`
2. 解压到任意位置（建议桌面或D盘）
3. 解压后会有一个 `dependencies_package` 文件夹

### 第二步：确保安装了Python
1. 检查电脑有没有安装Python 3.10或更高版本
2. 如果没有，去这里下载：https://www.python.org/downloads/
3. **安装时记得勾选"Add Python to PATH"**

### 第三步：启动程序
1. 进入 `dependencies_package` 文件夹
2. 双击 **`启动程序.bat`**
3. 等待程序启动即可

---

## ⚠️ 注意事项

- **可以解压到任意位置**（桌面、D盘、下载文件夹都可以）
- **可以移动文件夹**，移动后双击"启动程序.bat"依然能用
- **不要删除或修改 venv 文件夹**，里面的文件是依赖
- 如果杀毒软件提示，**允许运行**
- 首次启动可能需要10-20秒，请耐心等待

---

## 🔧 如果遇到问题

### 问题1：提示"Python is not installed"
**解决方法**：去 https://www.python.org/downloads/ 下载安装Python，记得勾选"Add Python to PATH"

### 问题2：提示"Virtual environment not found"
**解决方法**：重新下载完整的离线包，确保所有文件都在

### 问题3：程序打不开
**解决方法**：关闭杀毒软件再试试

---

## 📁 文件夹内容说明

```
dependencies_package/
├── venv/                  ← 所有的依赖都在这里（不要删除或修改）
├── run.py                 ← 控制台启动脚本
├── run.pyw                ← 无控制台启动脚本
├── requirements.txt       ← 依赖列表
└── 启动程序.bat           ← 推荐使用这个启动
```
