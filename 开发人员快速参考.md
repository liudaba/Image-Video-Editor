## 1. 快速上手

### ✅ 开发时怎么启动？
```
双击：启动主程序.bat
```

### 🚫 不要这样启动！
```
❌ 直接双击 run.py / run.pyw（不会激活虚拟环境 → 缺依赖）
```

## 2. 启动器说明

### 项目根目录（你开发的地方）
| 文件 | 用途 |
|------|------|
| **启动主程序.bat** | ⭐ 推荐！完整的环境检查 + 虚拟环境激活 |
| run.py | 有黑框启动，调试看输出用（需先激活venv） |
| run.pyw | 无黑框启动（需先激活venv） |
| start.bat | 简单快速启动（不推荐） |

### dependencies_package/（给客户的）
| 文件 | 说明 | 你要碰吗？ |
|------|------|----------|
| 整个文件夹 | 客户离线包 | ❌ 别碰！给客户的！ |

## 3. 打包发布

### 怎么打包？
```
双击：01build_exe.py（或运行 python 01build_exe.py）
```

### 打包后文件在哪里？
```
dist/短视频生成器/
├── 短视频生成器.exe  ← 主程序
├── 启动.vbs          ← 客户双击这个
└── ...其他
```

### 打包注意
- ✅ 只修改根目录代码
- ❌ 不要改 dependencies_package/（给客户的离线包，没用）
- ⚠️ 确保 config.json 里有 api_base_url

## 4. 常见问题

### Q1: 缺依赖？（提示缺少 moviepy）
A: 别直接双击 run.pyw！用 `启动主程序.bat`！

### Q2: 服务端在哪？
A: 在另一台服务器（8.141.101.155），项目根目录是 `videogen/`，不是这里！

### Q3: 想改代码怎么办？
A: 只改根目录的代码，dependencies_package/ 是给客户的！
