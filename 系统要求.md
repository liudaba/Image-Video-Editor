# 短视频生成器 - 系统要求

## 最低配置

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10 64位 或更高版本 |
| 内存 | 8 GB RAM |
| 磁盘空间 | 5 GB 可用空间（含语音模型） |
| 网络 | 首次使用需要联网（注册登录） |

## 推荐配置

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 11 64位 |
| 内存 | 16 GB RAM 或更高 |
| 显卡 | NVIDIA GeForce GTX 1060 6GB 或更高（本地画图模式） |
| 磁盘空间 | 10 GB 可用空间 |
| 网络 | 宽带网络连接 |

## 说明

- **无需Python环境**：软件已内置所有运行时，下载解压即可使用
- **无需NVIDIA显卡**：使用云端画图功能时不需要独立显卡，任何电脑都可以生成图片
- **本地画图模式**需要NVIDIA显卡并安装CUDA驱动，建议8GB以上显存
- **首次使用需要联网**：注册账号和登录验证需要网络连接
- **仅支持64位Windows系统**：不支持32位系统、macOS或Linux
