
隐藏控制台窗口修改说明
======================

1. 修改位置：My-Video Generator.py 第235-260行

2. 原始代码：
```python
if _is_pythonw or _has_no_console:
    ctypes.windll.kernel32.AllocConsole()
    _console_allocated = True

    hwnd = ctypes.windll.kernel32.GetConsoleWindow()

    if hwnd:
        user32 = ctypes.windll.user32

        hMenu = user32.GetSystemMenu(hwnd, False)
        if hMenu:
            SC_CLOSE = 0xF060
            MF_BYCOMMAND = 0x00000000
            user32.DeleteMenu(hMenu, SC_CLOSE, MF_BYCOMMAND)
            user32.DrawMenuBar(hwnd)

        GWL_STYLE = -16
        WS_SYSMENU = 0x00080000
        WS_MINIMIZEBOX = 0x00020000

        style = user32.GetWindowLongW(hwnd, GWL_STYLE)
        style = style & ~WS_SYSMENU
        style = style | WS_MINIMIZEBOX
        user32.SetWindowLongW(hwnd, GWL_STYLE, style)
        user32.DrawMenuBar(hwnd)
```

3. 修改后的代码：
```python
if _is_pythonw or _has_no_console:
    ctypes.windll.kernel32.AllocConsole()
    _console_allocated = True

    # 隐藏控制台窗口
    hwnd = ctypes.windll.kernel32.GetConsoleWindow()
    if hwnd:
        SW_HIDE = 0
        ctypes.windll.user32.ShowWindow(hwnd, SW_HIDE)
```

4. 修改说明：
   - 添加了隐藏控制台窗口的代码
   - 删除了修改控制台窗口样式的代码（因为窗口已经隐藏，不需要修改）
   - 保留了控制台窗口的分配和标题设置（如果需要）

5. 修改后的效果：
   - 程序启动时，控制台窗口将被隐藏
   - 用户将不会看到黑色的命令提示符窗口
   - 程序的日志输出仍然会重定向到控制台，但控制台窗口不可见
