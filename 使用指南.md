# 短视频生成器 - 使用指南

---

## 软件能做什么

把一段音频（比如配音、解说）放进去，软件自动帮你：
1. 把音频转成文字
2. 用AI给每句话配一张图
3. 把图片和音频合成短视频

---

## 安装与启动

### 解压
1. 下载压缩包，右键 → 解压到当前文件夹
2. 建议放到纯英文路径，比如 `D:\VideoGenerator\`
3. 不要放到含中文的路径下，可能出问题

### 启动
- **双击 `启动.vbs`** 打开软件
- 首次使用？先双击 `首次运行引导.bat`，自动帮你检查和修复环境

### 注册登录
1. 打开软件后弹出登录窗口
2. 点"注册"→ 填用户名和密码 → 注册
3. 自动登录，获得7天免费试用
4. 输入激活码可延长使用时间

## 画图功能配置

软件需要AI来画图，有两种方式，选一种就行。

### 方式一：在线画图（推荐）

不需要装任何额外软件，填个密钥就能用。

**操作步骤：**
1. 打开软件 → 点"设置"
2. 找到"云端画图"→ 打开开关
3. 选择服务商，填入API密钥
4. 保存设置，搞定

**去哪里拿密钥：**

| 服务商 | 注册地址 | 特点 |
|--------|---------|------|
| 硅基流动（推荐） | cloud.siliconflow.cn | 注册送免费额度，够试用 |
| 通义万相 | dashscope.aliyun.com | 阿里云出品，稳定 |
| DALL-E | platform.openai.com | 效果好，需海外账号 |

### 方式二：本地画图

需要电脑有独立显卡（建议8G显存以上），适合有经验的用户。

**需要安装：**
- Stable Diffusion WebUI — 画图软件，启动后软件会自动连接
- SD模型文件 — 放到SD的models目录下即可

**SD WebUI启动参数需加上：** `--api --listen`
这样软件才能连上SD。

## 文案功能配置

软件用AI帮你写视频文案和画图提示词，也有两种方式。

### 在线文案（推荐）

1. 打开软件 → 点"设置"
2. 找到"云端文案"→ 打开开关
3. 选择服务商，填入API密钥

**去哪里拿密钥：**

| 服务商 | 注册地址 | 特点 |
|--------|---------|------|
| DeepSeek（推荐） | platform.deepseek.com | 便宜好用 |
| 智谱GLM | open.bigmodel.cn | 有免费额度 |
| 通义千问 | dashscope.aliyun.com | 阿里云出品 |
| Kimi | platform.moonshot.cn | 长文本强 |

### 本地文案

安装 Ollama（ollama.com），下载一个模型（推荐 gemma3:4b），软件会自动检测。

## 制作视频的流程

1. **导入音频** — 支持mp3、wav等格式
2. **语音转文字** — 软件自动识别，内置语音模型，无需配置
3. **AI写文案** — 自动根据内容生成画图提示词
4. **AI画图** — 根据提示词自动生成每张配图
5. **合成视频** — 自动把图片和音频合成短视频，支持转场动画

## 常见问题

**问：软件打不开？**
→ 双击 `环境自检修复.bat` 检查问题
→ 杀毒软件拦截了？把整个文件夹加到信任列表
→ 文件被锁定？双击 `首次运行引导.bat` 自动解锁

**问：画图失败？**
→ 检查API密钥是否正确
→ 用本地SD的话，确认SD WebUI已经启动
→ 检查网络是否正常

**问：登录失败？**
→ 检查网络连接
→ 确认用户名密码没错

**问：软件卡住不动？**
→ 关掉重新打开
→ 还是不行就跑 `环境自检修复.bat`

**问：激活码怎么用？**
→ 打开软件 → 登录后 → 输入激活码 → 点激活

**问：想退款？**
→ 联系客服处理
