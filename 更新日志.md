# 短视频生成器 - 更新日志

## v2.0.0 (2026-05-16)

### 新功能
- 全新授权系统：支持注册登录、7天免费试用、激活码激活
- 云端画图：支持硅基流动等在线AI画图服务，无需本地显卡
- 云端文案：支持DeepSeek等在线AI文案生成
- 云端语音识别：支持在线语音转文字
- 多种视频转场效果：交叉淡化、缩放、滑动等
- 多种动画效果：缩放、平移、旋转等
- GPU硬件加速：支持NVIDIA NVENC、Intel QuickSync、AMD AMF硬件编码
- 桌面快捷方式创建工具

### 安全增强
- ECDSA非对称签名验证：防止授权数据被篡改
- 设备指纹注册限制：防止滥用试用账号
- 核心授权模块Cython编译：防止逆向工程

### 改进
- 优化视频渲染性能
- 优化内存管理
- 改进错误提示信息
- 增强心跳检测机制
