# 快速上手指南

> 把音频变成带画面的短视频，AI自动完成

---

## 本地模式 vs 云端模式

| | 本地模式 | 云端模式 |
|---|---|---|
| 需要安装 | SD WebUI + Ollama + 显卡 | 什么都不用装 |
| 费用 | 免费 | 几毛钱一条视频 |
| 适合 | 有显卡的进阶用户 | 新手、无显卡用户 |

> 三个功能（写文案、画图、语音识别）各有独立开关，可以混搭。比如：云端写文案 + 本地画图。

---

## 三步出视频

1. 双击 **`start.vbs`** 打开软件
2. 点「导入音频」选一个音频文件
3. 点「生成视频」，等几分钟就好

视频在 `output_project/` 文件夹里。

## 首次使用

### 解压
1. 右键压缩包 → 属性 → 勾选「解除锁定」（很重要！）
2. 用 7-Zip 或 WinRAR 解压到英文路径，如 `D:\VideoGenerator\`
3. 解压前先关杀毒软件，否则可能误删文件

### 启动
- 双击 **`start.vbs`** 启动
- 首次用？双击 **`FirstRunSetup.bat`**，自动检查环境

### 注册
- 打开软件 → 注册 → 填用户名、邮箱、密码（至少8位）
- 注册后自动获得15天免费试用

## 云端模式：填写 API Key

用云端模式只需填入 API Key，不用装任何东西。

### 怎么获取 API Key？

1. 去服务商网站注册账号
2. 进入「控制台」→「API Key管理」→「创建Key」
3. 复制 Key 保存好

| 用途 | 推荐服务商 | 网址 |
|------|-----------|------|
| AI写文案 | DeepSeek | platform.deepseek.com |
| AI写文案 | 智谱AI（有免费额度） | open.bigmodel.cn |
| AI画图 | 硅基流动（有免费额度） | cloud.siliconflow.cn |
| AI画图 | 通义万相 | dashscope.console.aliyun.com |

### 怎么填入软件？

1. 打开软件 → 点「高级设置」
2. 勾选「启用云端LLM」→ 选服务商 → 粘贴Key → 选模型
3. 勾选「启用云端生图」→ 选服务商 → 粘贴Key → 选模型
4. 点「测试连接」确认能用 → 点「应用设置」

> Key 在本地加密保存，不会上传。一条3分钟视频大约花几毛钱。

## 本地模式：安装软件

### 画图 — SD WebUI
1. 下载 SD WebUI（搜"秋叶SD启动器"最简单）
2. 启动时必须加 `--api` 参数（秋叶启动器里勾选"开启API"即可）
3. 启动后软件会自动检测

### 写文案 — Ollama（可选）
1. 去 ollama.com 下载安装
2. 命令行执行 `ollama pull gemma3:4b`
3. 软件会自动检测

## 常见问题

| 问题 | 解决 |
|------|------|
| 打不开 | 运行 `CheckEnv.bat`；关杀毒重新解压；右键以管理员运行 |
| 文件不全 | 用7-Zip重新解压；先右键压缩包→属性→解除锁定 |
| 画图失败 | 检查API Key对不对（点测试连接）；本地SD要加 --api 参数 |
| 登录失败 | 检查网络；忘记密码点「忘记密码?」 |
| 云端报错 | 检查Key对不对；检查账户余额 |
