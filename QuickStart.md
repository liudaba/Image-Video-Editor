# 快速上手指南

> 音频放进去，视频出来 —— AI全自动完成

---

## 第一步：解压

1. 右键压缩包 → **属性** → 最下方勾选**「解除锁定」** → 确定
   > 这一步很重要！不做可能无法运行
2. 用 **7-Zip** 或 **WinRAR** 解压到**纯英文路径**，例如 `D:\VideoGenerator\`
   > 路径中不要有中文或空格，否则可能出问题
3. 如果杀毒软件弹出警告，请选择**允许/信任**

## 第二步：启动

- 双击 **`start.vbs`** 即可启动（推荐）
- 首次使用？双击 **`FirstRunSetup.bat`**，自动检查环境并启动

## 第三步：出视频

1. 点「导入音频」选一个音频文件（mp3、wav等）
2. 点「生成视频」
3. 等几分钟，视频自动保存在 `output_project/` 文件夹里

> 就这么简单！三步出视频

| | 云端模式（推荐新手） | 本地模式（需显卡） |

| 需要安装 | 什么都不用装 | SD WebUI + Ollama |

## 云端模式：填 API Key

云端模式只需填入 API Key，不用装任何东西。

### 怎么获取 API Key？

1. 去服务商网站注册账号
2. 进入「控制台」→「API Key管理」→「创建Key」
3. 复制 Key 保存好

| 用途 | 推荐服务商 | 网址 |
|------|-----------|------|
| AI写文案 | DeepSeek | platform.deepseek.com |
| AI写文案 | 智谱AI（有免费额度） | open.bigmodel.cn |
| AI画图 | 硅基流动（有免费额度） | cloud.siliconflow.cn |

### 怎么填入软件？

1. 打开软件 → 点「高级设置」
2. 勾选「启用云端LLM」→ 选服务商 → 粘贴Key → 选模型
3. 勾选「启用云端生图」→ 选服务商 → 粘贴Key → 选模型
4. 点「测试连接」确认能用 → 点「应用设置」

> Key 在本地加密保存，不会上传。一条3分钟视频大约花几毛钱。

## 本地模式：安装软件

### 画图 — SD WebUI（需显卡）

1. 下载 SD WebUI（搜"秋叶SD启动器"最简单）
2. 启动时必须加 `--api` 参数（秋叶启动器里勾选"开启API"即可）
3. 启动后软件会自动检测

### 写文案 — Ollama（可选）

1. 去 ollama.com 下载安装
2. 命令行执行 `ollama pull gemma3:4b`
3. 软件会自动检测

## 常见问题

| 问题 | 解决 |
|------|------|
| 打不开 | 运行 `CheckEnv.bat` 检查；右键以管理员运行 |
| 文件不全 | 用7-Zip重新解压；先右键压缩包→属性→解除锁定 |
| 画图失败 | 检查API Key对不对（点测试连接）；本地SD要加 --api 参数 |
| 登录失败 | 检查网络；忘记密码点「忘记密码?」 |
| 云端报错 | 检查Key对不对；检查账户余额 |
