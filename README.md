1. 打开软件后会弹出登录窗口
2. 点"注册"标签页，填用户名、邮箱和密码

## 💡 两种使用模式

### 模式1：云端模式（推荐新手 ⭐）

**优点：**
- ✅ 无需安装任何AI软件
- ✅ 对电脑配置要求低（能上网就行）
- ✅ 速度快、质量高

**需要准备：**
- 联网
- API Key（见下方获取教程，5分钟搞定）

**设置方法：**
1. 点击 **"⚙️ 高级设置"**
2. 勾选 **"启用云端LLM"**（AI写文案）
3. 勾选 **"启用云端生图"**（AI画图）
4. 可选：勾选 **"启用云端语音识别"**（AI听写音频，替代本地Whisper）
5. 填写API Key（推荐使用 DeepSeek 或硅基流动）
6. 点击 **"✅ 应用设置"**

> 💡 **首次使用？** 查看 [如何获取API Key](#如何获取api-key云端模式必需)

---

### 模式2：本地模式（适合有GPU的用户）

**优点：**
- ✅ 完全免费（不消耗API额度）
- ✅ 数据不出电脑，隐私安全

**需要准备：**
- NVIDIA显卡（GTX 1060及以上）
- 安装 Ollama（AI大脑，写文案用）
- 安装 SD WebUI（AI画笔，画图用）

**安装教程：**

- SD WebUI：搜索 "Stable Diffusion WebUI 安装教程"

> ⚠️ **注意**：SD WebUI 启动时需要加 `--api` 参数

## ❓ 常见问题（FAQ）

### 🚨 启动问题

#### Q1：双击启动文件没反应？
**解决方法：**
1. 确认已解压到**纯英文路径**（如 `D:\VideoGenerator\`），不要放桌面或含中文的路径
2. 杀毒软件可能拦截了程序，把整个文件夹加到**信任列表**
3. 双击 `检查环境.bat` 检查运行环境
4. 右键点击启动文件，选择 **"以管理员身份运行"**

#### Q2：提示"Python未安装"？
**解决方法：**
1. 访问 https://www.python.org/downloads/
2. 下载 Python 3.10 或更高版本
3. 安装时**务必勾选** "Add Python to PATH"
4. 安装完成后重新运行启动文件

#### Q3：提示"依赖安装失败"？
**解决方法：**
1. 确保网络连接正常
2. 右键点击启动文件
3. 选择 **"以管理员身份运行"**
4. 等待依赖安装完成

#### Q4：提示"Ollama未连接"？
**解决方法（二选一）：**
- **方案A**：安装并启动Ollama（本地模式）
- **方案B**：在高级设置中启用云端LLM（推荐新手，无需安装Ollama）

---

### 🖼️ 生成问题

#### Q5：生成图片时卡住不动？
**检查清单：**
- ✅ 是否启用了云端生图？（推荐，无需本地显卡）
- ✅ 如果使用本地SD，确认SD WebUI已启动且加了 `--api` 参数
- ✅ API密钥是否填对？点"测试连接"确认
- ✅ 网络是否正常？

#### Q6：生成的图片和音频内容对不上？
**优化方法：**
1. 在"高级设置" → "主题自定义"中，手动填写核心主题（越具体越好）
   - ❌ 不好：`科学`
   - ✅ 好：`人类与病毒的共存关系`
2. 尝试切换更强的云端模型

#### Q7：视频合成失败？
**检查：**
- ✅ 原始音频文件是否还在
- ✅ 磁盘空间是否充足（至少5GB）
- ✅ 图片是否全部生成完成

**解决方法**：点击垃圾桶🗑️清除数据，重新导入音频

#### Q8：如何重新开始？
点击音频旁边的 **🗑️ 垃圾桶按钮**，清除当前数据后重新导入音频即可

> 💡 旧文件会移到"垃圾桶"文件夹，可以手动找回

## ⚠️ 注意事项

1. **一次只处理一个音频** — 不要同时运行多个程序实例
2. **音频质量很重要** — 录音越清晰，AI分析越准确，效果越好
3. **不要中途关闭程序** — 生成过程中关闭可能导致文件损坏
4. **定期清理垃圾桶** — 垃圾桶不会自动清空，占用磁盘空间
5. **保持服务运行** — 生成期间不要关闭SD WebUI（本地模式）
6. **首次使用需联网** — 注册登录和授权验证需要网络连接
