## ✨ 功能介绍

### 这个工具能做什么？
✅ **自动语音识别** - 精准提取音频中的文字内容  
✅ **智能分镜生成** - AI根据内容自动划分视频场景  
✅ **AI图像生成** - 为每个场景自动生成匹配的画面  
✅ **专业视频合成** - 自动添加转场、字幕、背景音乐  
✅ **主题一致性检查** - 确保整个视频风格统一协调  

### 适用场景
- 📚 知识科普视频
- 📰 新闻资讯配音
- 🎵 音乐MV制作
- 📖 有声书可视化
- 🎓 在线教育课程

## 🚀 快速开始（3步搞定）

### 第1步：启动程序
双击运行 **`启动程序.bat`**，等待程序加载完成（约30秒）

看到日志显示 **"✅ 程序启动完成，工具已就绪！"** 即可开始使用

### 第2步：导入音频
点击左侧面板的 **"📂 导入音频"** 按钮，选择你的音频文件（支持 .wav/.mp3/.m4a）

### 第3步：生成视频
点击右侧面板的 **"🎞️ 跑图生成视频"** 按钮，程序会自动完成：
1. 分析音频内容
2. 生成分镜脚本
3. 生成场景图片
4. 合成最终视频

⏱️ **预计耗时**：59个分镜约需15-20分钟（取决于电脑配置）

## ❓ 常见问题

### Q1: 程序启动很慢怎么办？
**A**: 首次启动需要加载AI模型（约30秒），这是正常现象。后续启动会更快。

**优化建议**：
- 确保电脑有独立显卡（NVIDIA RTX系列最佳）
- 关闭其他占用GPU的程序
- 耐心等待"程序启动完成"提示

---

### Q2: 生成图片时卡住了？
**A**: 可能的原因和解决方案：

**原因1：SD服务未启动**
- 检查是否已启动 Stable Diffusion WebUI
- 确认地址是 `http://localhost:7860`

**原因2：显存不足**
- 关闭浏览器和其他GPU程序
- 降低图片分辨率（在高级设置中调整为 512x512）

**原因3：网络问题**
- 检查SD API连接状态
- 重启SD WebUI服务

---

### Q3: 生成的图片与音频内容不符？
**A**: 使用手动模式重新生成分镜：

1. 点击 **"🎬 生成分镜脚本"**
2. 观察日志中的"主题一致性检查"结果
3. 如果偏离率 > 15%，程序会自动修正
4. 修正后重新生成图片

**提示**：可以在高级设置中自定义核心主题和视觉基调

---

### Q4: 视频合成失败？
**A**: 检查以下几点：

✅ 是否已生成所有图片（59张）  
✅ 图片文件名是否正确（shot_001.png ~ shot_059.png）  
✅ 音频文件是否存在且未被删除  
✅ 磁盘空间是否充足（至少5GB）  

**解决方法**：
- 删除 `output_project/` 文件夹重新开始
- 确保图片和音频文件完整

---

### Q5: 如何调整视频质量？
**A**: 在左侧面板的 **"高级设置"** 中可以调整：

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 图片尺寸 | 768x512 | 越大越清晰，但速度越慢 |
| SD采样步数 | 25 | 越高画质越好（20-30） |
| CFG Scale | 7.0 | 控制创意程度（5-10） |
| 图像生成线程 | 12 | 根据显存调整（8-16） |

---

### Q6: 可以中途暂停吗？
**A**: 可以！程序支持断点续传：

- **生成分镜时**：无法暂停（必须完成）
- **生成图片时**：可以点击"暂停"，稍后继续
- **合成视频时**：建议不要中断

**注意**：暂停后再次点击相同按钮会继续未完成的任务

---

### Q7: 为什么有些分镜被标记为"偏离主题"？
**A**: 这是正常的AI生成特性：

- **偏离率 < 15%**：可接受范围，不影响整体质量
- **偏离率 ≥ 15%**：程序会自动调用AI重新生成提示词

**智能翻译机制**：
程序采用双层翻译策略确保主题匹配准确：
1. 📚 **核心字典**：高频词汇快速翻译（30+常用词，<1毫秒）
2. 🤖 **LLM智能翻译**：生僻词汇自动调用AI翻译，结果自动缓存
3. ♻️ **自动学习**：新词汇翻译后自动加入缓存，下次直接使用

**你不需要手动维护翻译字典！** 系统会自动学习和记忆新词汇。

---

### Q8: 生成的视频没有声音？
**A**: 检查音频文件：

✅ 音频格式是否正确（.wav/.mp3/.m4a）  
✅ 音频文件是否损坏（用播放器测试）  
✅ 音量是否正常（不要太低）  

**解决方法**：
- 重新导入音频文件
- 使用音频编辑软件调整音量
- 确保音频时长与分镜总时长匹配

## ⚠️ 注意事项

### 硬件要求
- **最低配置**：
  - CPU: Intel i5 / AMD Ryzen 5
  - 内存: 8GB
  - 显卡: NVIDIA GTX 1060 (4GB显存)
  - 硬盘: 至少10GB可用空间

- **推荐配置**：
  - CPU: Intel i7 / AMD Ryzen 7
  - 内存: 16GB
  - 显卡: NVIDIA RTX 3060 (8GB显存)
  - 硬盘: SSD固态硬盘

### 软件依赖
程序会自动检查并提示缺少的组件：
- ✅ Python 3.10+
- ✅ Ollama（本地大模型服务）
- ✅ Stable Diffusion WebUI（图像生成）
- ✅ Whisper模型（语音识别）

### 使用建议
1. **首次使用前**：
   - 阅读本说明书
   - 准备一段3-5分钟的测试音频
   - 确保所有服务已启动

2. **批量处理时**：
   - 一次只处理一个音频文件
   - 完成后清理 `output_project/` 文件夹
   - 避免同时运行多个实例

3. **质量保证**：
   - 音频清晰度越高，效果越好
   - 建议使用普通话或标准英语
   - 避免背景噪音过大的音频

4. **文件管理**：
   - 定期备份重要的分镜脚本
   - 不要手动修改 `shots_data.json`
   - 视频生成后可删除图片文件夹节省空间

## 🎯 最佳实践

### 获得高质量视频的技巧

1. **音频准备**：
   - 使用清晰的录音，避免回声和噪音
   - 语速适中（每分钟150-200字）
   - 提前修剪掉开头和结尾的空白

2. **主题设置**：
   - 在高级设置中指定明确的核心主题
   - 选择合适的视觉基调（严肃/轻松/紧张等）
   - 提供具体的关键词（如"东京街头""都市女性"）

3. **图片质量**：
   - 使用高分辨率SD模型
   - 增加采样步数到30（牺牲速度换质量）
   - 选择合适的提示词类型（ARV写实/动漫等）

4. **后期处理**：
   - 视频生成后用剪辑软件微调
   - 添加片头片尾
   - 调整音量和背景音乐
