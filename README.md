# 短视频生成器 - 使用说明书

## 这是什么？

这是一个**自动生成短视频**的工具。你只需要：
1. 准备一段音频（比如讲故事、解说、配音）
2. 点击几个按钮
3. 程序就会自动帮你生成一个带画面的视频

**适合谁用？**
- 想做短视频但不会剪辑的人
- 想把音频变成视频的人
- 想批量生成视频内容的人

## 第二步：安装必要的软件

### 必须安装的软件

#### 1. Python（程序的运行的基础）

**检查是否已经安装：**
- 按 `Win + R` 键，输入 `cmd`，回车
- 输入 `python --version`，回车
- 如果显示版本号（比如 Python 3.11.0），说明已安装，跳到下一步
- 如果显示"不是内部或外部命令"，说明没安装，继续看下面

**安装方法：**
1. 打开 https://www.python.org/downloads/
2. 点击 **"Download Python 3.11.x"**（最新的3.11版本）
3. 下载后双击安装
4. **重要**：安装时勾选 **"Add Python to PATH"**，然后点击 Install Now
5. 等待安装完成

#### 2. FFmpeg（处理视频必需的）

**这是什么？** 这是一个处理视频的工具，我们的程序需要它来生成视频。

**安装方法：**

**Windows用户：**
1. 打开 https://github.com/BtbN/FFmpeg-Builds/releases
2. 找到最新版本，下载 `ffmpeg-master-latest-win64-gpl.zip`
3. 解压到 `C:\ffmpeg` 文件夹
4. 设置环境变量：
   - 右键"此电脑" → 属性 → 高级系统设置 → 环境变量
   - 在"系统变量"中找到 Path，双击打开
   - 点击"新建"，输入 `C:\ffmpeg\bin`
   - 点击确定保存

**验证安装：**
- 打开cmd，输入 `ffmpeg -version`
- 如果显示版本信息，说明安装成功

## 第三步：安装Python依赖包

**什么是依赖包？** 就是程序运行需要的其他工具包。

**安装方法：**

1. 打开cmd（按 `Win + R`，输入 `cmd`，回车）
2. 进入项目文件夹：
   ```bash
   cd C:\Users\你的用户名\Desktop\短视频生成器
   ```
   （注意：根据你实际放的位置修改路径）
3. 输入以下命令，回车：
   ```bash
   pip install -r requirements.txt
   ```
4. 等待安装完成（可能需要几分钟，取决于网速）

**如果安装很慢或失败：**
- 可以尝试使用国内镜像：
  ```bash
  pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
  ```

## 第四步：下载AI模型文件

**什么是模型文件？** 这是AI的大脑，用来识别语音、生成图片等。

**注意：** 这些文件很大（共约800MB），所以没放在Git仓库里，需要你自己下载。

### 需要下载的模型：

#### 1. Whisper模型（语音识别用）

**文件名：** `tiny.pt` 或 `medium.pt`

**下载地址：**
- 程序第一次运行时会自动下载（推荐）
- 或者手动下载：https://huggingface.co/openai/whisper-medium

**放哪里：** 下载后放到 `models/` 文件夹里

#### 2. LatentSync模型（口型同步用，可选）

**文件名：** `latentsync_unet.pt`

**说明：** 这个模型用于让视频人物的口型和音频对齐。如果你只是做图片视频，不需要这个。

**下载地址：** 需要从其他渠道获取（文件太大，约700MB）

**放哪里：** 下载后放到 `models/` 文件夹里

## 第六步：使用教程

### 基本操作流程

#### 1. 导入音频

- 点击界面上的 **"导入音频"** 按钮
- 选择你的音频文件（支持 mp3, wav, ogg 格式）
- 等待音频加载完成

#### 2. 生成分镜脚本

- 点击 **"一键生成分镜"** 按钮
- 程序会自动：
  1. 识别音频中的语音内容
  2. 根据内容生成画面描述
  3. 计算每个画面显示的时间
- 等待生成完成（可能需要几分钟）
- 生成的分镜脚本会显示在界面上

#### 3. 生成图片

**方法A：用AI自动生成（需要配置Stable Diffusion）**
- 在界面上配置 Stable Diffusion API 地址
- 点击生成图片按钮
- 等待AI生成图片

**方法B：自己准备图片（推荐新手用）**
- 打开项目文件夹里的 `images/` 目录（如果没有就新建一个）
- 把你准备好的图片放进去
- 图片命名格式：`001.png`, `002.png`, `003.png`...（对应第1个、第2个、第3个分镜）

#### 4. 生成视频

- 点击 **"生成视频"** 按钮
- 程序会把图片和音频合成为一个视频
- 生成的视频会保存在 `output_project/` 文件夹里

```
┌─────────────────────────────────────┐
│  ① 音频导入区                        │
│     [导入音频按钮]  [音频信息]        │
├─────────────────────────────────────┤
│  ② 分镜脚本区（显示生成的画面描述）   │
│     [一键生成分镜按钮]                │
├─────────────────────────────────────┤
│  ③ 图片生成区                        │
│     [生成图片按钮]  [图片预览]        │
├─────────────────────────────────────┤
│  ④ 视频生成区                        │
│     [生成视频按钮]  [进度显示]        │
├─────────────────────────────────────┤
│  ⑤ 设置区                            │
│     [过渡模式] [分辨率] [其他设置]    │
└─────────────────────────────────────┘
```

### 重要设置项

#### 过渡模式（在设置区选择）

- **硬切**（推荐）：直接切换画面，速度最快
- **淡入淡出**：画面慢慢出现/消失，较慢
- **交叉溶解**：两个画面叠加过渡，较慢

**建议：** 如果追求速度，选"硬切"；如果想要好看的效果，选"淡入淡出"

#### 分辨率设置

- **默认**：1920x1080（1080P高清）
- **可选**：根据你的需要调整
- **建议**：保持默认即可

## 常见问题解答

### Q1: 程序打不开怎么办？

**可能原因和解决方法：**

1. **提示"找不到Python"**
   - 原因：Python没安装或没添加到PATH
   - 解决：重新安装Python，记得勾选"Add Python to PATH"

2. **提示"缺少xxx模块"**
   - 原因：依赖包没安装完整
   - 解决：重新运行 `pip install -r requirements.txt`

3. **双击没反应**
   - 原因：可能是Python路径问题
   - 解决：尝试用cmd命令行运行，看有什么错误提示

### Q2: 生成视频很慢怎么办？

**加速方法：**

1. **使用"硬切"过渡模式**（默认就是）
2. **关闭动画效果**（在设置里选"无"）
3. **使用GPU加速**（如果有NVIDIA显卡）
4. **减少分镜数量**（每个分镜都会生成一张图，图越多越慢）

### Q3: 音画不同步怎么办？

**检查以下几点：**

1. 确保使用的是最新版本代码
2. 分镜生成时音频要完整识别
3. 图片数量要和分镜数量一致
4. 如果还是不同步，尝试重新生成分镜

### Q4: 生成的图片质量不好怎么办？

**改善方法：**

1. **使用Stable Diffusion本地部署**：自己搭建SD，画质更好控制
2. **调整提示词**：在分镜脚本里修改画面描述，越详细越好
3. **使用更好的模型**：比如用SDXL代替SD 1.5

### Q5: 可以批量生成吗？

**目前版本：** 一次只能处理一个音频文件

**变通方法：**
- 准备多个音频文件
- 一个一个导入生成
- 或者同时打开多个程序窗口（如果电脑配置够好）

### Q6: 模型文件下载太慢/失败怎么办？

**解决方法：**

1. **Whisper模型**：让程序自动下载，第一次用时会自动下载
2. **LatentSync模型**：
   - 可以从百度网盘等国内镜像下载
   - 或者暂时不用口型同步功能（不影响主要功能）

## 项目文件说明

下载后你会看到以下文件：

```
短视频生成器/
├── My-Video Generator.py    ← 主程序（不要删）
├── config.json              ← 配置文件（保存你的设置）
├── requirements.txt         ← 依赖列表（安装依赖用的）
├── README.md                ← 这个说明书
├── 启动.bat                 ← Windows启动脚本
├── 启动.vbs                 ← Windows无窗口启动脚本
├── .gitignore              ← Git忽略配置（不用管）
├── models/                 ← 模型文件夹（放AI模型）
│   └── README.md          ← 模型说明
└── output_project/        ← 输出文件夹（生成的视频会在这里）
```

**不要删除的文件：**
- `My-Video Generator.py`（主程序）
- `config.json`（你的设置）
- `requirements.txt`（依赖列表）

**可以删除的文件：**
- 生成的视频文件（在output_project里）
- 临时文件
