## 这个工具是干什么的？

简单说：**你给它一段音频（比如一段朗读、一段解说），它就能自动生成一个带画面的短视频。**

它是怎么做到的？分4步：

```
音频 → 语音识别（把声音变成文字）→ 分镜脚本（把文字切成一个个场景）→ AI画图（给每个场景画一张图）→ 合成视频（图片+音频=视频）
```

**适合做什么类型的视频？**
- 📚 知识科普（比如"宇宙是怎么来的"）
- 📖 有声书可视化（给小说配画面）
- 📓 教育课程（给讲解配图）
- 📰 新闻资讯配音视频
- 🎵 音乐MV

## 两种AI大脑模式

本工具使用 **3类AI模型** 协同工作，其中"AI思考大脑"支持本地/云端两种模式：

### 🧩 三类AI模型分工

| 模型类型 | 职责 | 默认运行位置 | 云端可替代？ | 云端替代后 |
|----------|------|-------------|-------------|-----------|
| **Whisper 语音识别** | 把音频转成文字 | 本地 | ✅ 可替代 | 使用OpenAI Whisper API，无需本地GPU |
| **LLM 大语言模型** | 内容分析、提示词生成、语音纠错等"思考"任务 | 本地Ollama | ✅ 可替代 | 使用云端API（DeepSeek/GPT-4o等） |
| **Stable Diffusion 制图** | 根据提示词生成场景图片 | 本地（SD WebUI） | ✅ 可替代 | 使用云端生图API（硅基流动/DALL-E等） |

> 💡 **简单理解**：Whisper是"耳朵"（听音频），LLM是"大脑"（思考分析），SD是"画笔"（画图）。云端可以替代"耳朵"、"大脑"和"画笔"中的任何一个或全部。

### 🖥️ 本地模式（默认）

使用本地安装的 **Ollama** 大模型作为"AI思考大脑"，完全免费，无需联网。

- ✅ 完全免费，不花一分钱
- ✅ 数据不出电脑，隐私安全
- ⚠️ 需要本地安装Ollama并下载模型
- ⚠️ 依赖本地显卡/CPU算力，速度受硬件影响

### ☁️ 云端模式

连接云端收费大模型API（如DeepSeek、通义千问等）作为"AI思考大脑"，按调用量付费。

- ✅ 无需本地安装Ollama，省去配置麻烦
- ✅ 云端算力强大，生成速度更快、质量更高
- ✅ 支持多种顶级模型（GPT-4o、DeepSeek-V3等）
- ⚠️ 需要API Key，按使用量付费（费用很低）
- ⚠️ 需要联网

> 💡 **两种模式互斥**：启用云端LLM后，所有AI思考任务全部由云端完成，不使用本地Ollama；关闭后全部由本地完成。不会混用。

> 💡 **三个云端开关独立控制**：云端LLM、云端ASR、云端生图是三个独立开关，可以分别启用或关闭。例如可以只用云端LLM但保留本地Whisper和SD，或者三者都用云端（完全无需本地AI模型）。

> ⚠️ **注意**：启用云端生图后，本地SD WebUI不再需要运行。但如果不启用云端生图，SD WebUI仍然必须本地运行。

### 必须安装的软件

| 软件 | 干什么用的 | 怎么安装 |
|------|-----------|----------|
| **Python 3.10+** | 运行本程序 | 去 python.org 下载安装 |
| **Stable Diffusion WebUI** | AI画图，用来生成场景图片 | 搜索"Stable Diffusion WebUI 安装教程"，启动时需要加 `--api` 参数 |

### 大模型选择

你需要为三类AI任务选择运行方式：

| AI任务 | 本地方式 | 云端方式 | 说明 |
|--------|---------|---------|------|
| **AI思考大脑**（内容分析、提示词生成） | Ollama（免费，需安装） | 云端大模型API（付费，无需安装） | 二选一，启用云端后Ollama自动禁用 |
| **语音识别**（音频转文字） | Whisper（本地，需GPU） | OpenAI Whisper API（云端，无需GPU） | 二选一，启用云端后Whisper自动禁用 |
| **AI制图**（生成场景图片） | SD WebUI（本地，需GPU） | 云端生图API（付费，无需GPU） | 二选一，启用云端后本地SD自动禁用 |

> 💡 **推荐组合**：
> - **省钱组合**：全部本地（需要好的GPU）
> - **省GPU组合**：云端LLM + 云端ASR + 本地SD（Ollama和Whisper的GPU显存释放给SD画图）
> - **全云端组合**：三个都启用云端（完全无需本地AI模型，只需运行程序本身）
> - **灵活组合**：按需选择，三个开关独立控制

### 首次运行

1. 双击 **`检查环境.bat`**，检查Python和依赖是否安装完整
2. 如果提示缺少依赖包，在命令行运行：`pip install -r requirements.txt`

4. 确保Stable Diffusion WebUI正在运行（加 `--api` 参数）

### 第1步：启动程序

双击 **`start.bat`**

等待约30秒，当日志区域显示 **"✅ 程序启动完成，工具已就绪！"** 就可以开始了

## 详细操作步骤

### 方式一：一键模式（推荐新手）

**最简单的用法，只需要2次点击：**

```
导入音频 → 点击"🎞️ 生成视频" → 等待完成
```

程序会自动判断当前状态，智能执行：
- 如果没有分镜脚本 → 先生成分镜，再画图，再合成
- 如果有分镜但没有图片 → 先画图，再合成
- 如果分镜和图片都有 → 直接合成视频

### 方式二：分步模式（推荐进阶用户）

**想要更多控制力？可以分步操作：**

#### 第1步：生成分镜脚本

点击 **"🎬 生成分镜脚本"**

程序会自动完成4件事：

1. **语音识别** — 用Whisper把音频转成文字（带时间戳）
2. **内容分析** — 用大模型分析文章主题、情感、视觉基调
3. **语音纠错** — 自动纠正语音识别中的常见错误（如同音字、专有名词）
4. **生成提示词** — 为每个场景生成AI画图的提示词（两阶段思考：先理解语义，再翻译为视觉场景）

> 💡 分镜脚本保存在 `output_project/shots_data.json`

#### 第2步：生成视频

点击 **"🎞️ 生成视频"**

程序会自动完成：
1. 连接SD API生成场景图片
2. 把图片和音频合成为视频

> 💡 生成的图片在 `output_project/images/` 文件夹，视频在 `output_project/` 文件夹

### 关于垃圾桶功能 🗑️

程序有一个**安全垃圾桶**机制，防止误删文件：

- 每次清理数据时，旧文件不会直接删除，而是移到"垃圾桶"文件夹
- 垃圾桶按时间分类（如 `清理_20260427_203000`），方便找回
- 如果误删了重要文件，可以从垃圾桶里手动复制回来
- 垃圾桶不会自动清空，需要手动清理

## 云端大模型设置详解

在高级设置面板中，有 **三个独立的云端开关**，可以分别控制：

### 核心逻辑

```
☁️ 云端LLM开关（控制"AI思考大脑"）
  未启用 → AI思考任务由本地Ollama完成
  已启用 → AI思考任务由云端大模型完成，Ollama模型选择自动禁用

☁️ 云端ASR开关（控制"AI语音识别"）
  未启用 → 语音识别由本地Whisper完成
  已启用 → 语音识别由OpenAI Whisper API完成，Whisper模型选择自动禁用

☁️ 云端生图开关（控制"AI画笔"）
  未启用 → 图片由本地SD WebUI生成
  已启用 → 图片由云端生图API生成，本地SD设置自动禁用
```

> ⚠️ 三个云端开关**独立控制、互不影响**。你可以自由组合：
> - 全部本地（默认，免费但需要GPU）
> - 只启用云端LLM（省去Ollama安装，保留本地Whisper和SD）
> - 云端LLM + 云端ASR（Ollama和Whisper的GPU显存释放给SD画图）
> - 云端LLM + 云端生图（Ollama和SD都不需要本地运行）
> - **全云端**：三个都启用（完全无需本地AI模型，只需运行程序本身）

## 常见问题

### Q1：程序启动后弹窗说"Ollama服务未连接"？

**原因**：Ollama没有运行，且未启用云端模式

**解决方法（二选一）**：
- **方案A（本地模式）**：打开Ollama应用程序，等待启动完成，重新启动本程序
- **方案B（云端模式）**：在高级设置中启用云端大模型，配置API Key后即可不依赖Ollama

> 💡 如果已启用云端模式，Ollama连接失败的提示可以忽略

### Q2：生成图片时一直卡住不动？

**可能原因和解决方法**：

| 原因 | 解决方法 |
|------|----------|
| SD WebUI没启动 | 启动Stable Diffusion WebUI（记得加 `--api` 参数） |
| SD API地址不对 | 在高级设置中检查API地址是否为 `http://127.0.0.1:8080` |
| 显存不足 | 关闭浏览器等占显存的程序，或降低图片分辨率 |
| SD正在处理中 | 耐心等待，SD生成一张图需要几秒到十几秒 |

### Q3：生成的图片和音频内容对不上？

**解决方法**（按优先级排序）：
1. 在高级设置的"主题自定义"中，手动填写核心主题和视觉基调（越具体越好，如"人类与病毒的微妙共存关系"而非"科学"）
2. 尝试切换提示词类型（SD提示词 ↔ ARV写实提示词）
3. 尝试更换大模型：
   - 本地模式：更换Ollama模型（推荐 `gemma3:4b` 或 `qwen3:8b`）
   - 云端模式：切换更强的模型（如 DeepSeek-V3 → GPT-4o）
4. 确认制图模型选择正确——下拉菜单中的类型标签（[SD1.5]/[SDXL]/[Flux]/[SD3]）是否与实际模型匹配

> 💡 提示词生成系统已优化：自动将抽象概念转化为视觉隐喻（如"利弊"→天平，"克制"→留白），禁止ICU/医疗场景泛滥，自动修复SD语法错误。填写具体的主题自定义内容可进一步提升准确度。

### Q4：视频合成失败？

**检查清单**：
- ✅ 音频文件是否还在（不要删除导入的音频文件）
- ✅ 磁盘空间是否充足（至少留5GB）
- ✅ 图片是否全部生成完成（看日志有没有报错）

**解决方法**：点击垃圾桶🗑️清除所有数据，重新导入音频从头开始

### Q5：程序运行很慢？

**优化建议**：
- 关闭其他占用GPU的程序（浏览器、游戏等）
- 在高级设置中降低图片分辨率（如 1280x720）
- 降低分镜创建线程数（减少显存占用）
- 使用更小的Whisper模型（如 small 或 base）
- **启用云端大模型**：云端模型速度通常比本地快很多

### Q6：生成的视频没有声音？

**检查**：
- 原始音频文件是否正常（用播放器试听一下）
- 音频格式是否支持（.mp3 .wav .m4a .flac）
- 音频音量是否太低

### Q7：如何重新开始？

点击音频状态旁边的 **🗑️ 垃圾桶按钮**，会清除当前音频和所有相关数据，然后重新导入音频即可

> 💡 旧文件会移到垃圾桶文件夹，不会直接删除，需要的话可以从垃圾桶找回

### Q8：下载了新的SD模型，程序里看不到？

**解决方法**：
1. 确认新模型已放入SD WebUI的 `models/Stable-diffusion/` 目录
2. 确认SD WebUI正在运行
3. 在高级设置的模型下拉菜单旁，点击 **🔄 刷新按钮**
4. 下拉菜单会自动更新，新模型会显示类型标签（如 `[SDXL] 新模型名`）

> 💡 程序会自动识别模型类型（SD1.5/SDXL/Flux/SD3），并自动切换对应的提示词模板和生成参数

### Q9：云端大模型调用失败？

**检查清单**：
- ✅ API Key是否正确（没有多余空格）
- ✅ 网络是否正常（云端模型需要联网）
- ✅ 账户余额是否充足
- ✅ 点击"测试连接"确认服务可用

**解决方法**：
- 如果是网络问题，检查代理设置
- 如果是余额不足，到服务商官网充值
- 如果想临时切回本地模式，取消勾选"启用云端模型"即可

## 注意事项

### 使用建议

1. **一次只处理一个音频** — 不要同时运行多个程序实例
2. **音频质量很重要** — 录音越清晰，识别越准确，效果越好
3. **不要中途关闭程序** — 视频生成过程中关闭程序可能导致文件损坏
4. **定期清理垃圾桶** — 垃圾桶不会自动清空，久了会占硬盘空间
5. **保持服务运行** — 生成视频期间不要关闭SD WebUI
