# 短视频生成器 - 使用说明

---

## 第一步：解压

1. 下载压缩包
2. 右键 → 解压到当前文件夹
3. 建议解压到纯英文路径，比如 `D:\VideoGenerator\`
4. 不要解压到桌面或含中文的路径，可能出问题

## 第二步：启动

**双击 `启动.vbs`** 即可打开软件

> 首次打开？建议先双击 `首次运行引导.bat`，会自动帮你检查环境

## 第三步：注册登录

1. 打开软件后会弹出登录窗口
2. 点"注册"，填用户名和密码
3. 注册完自动登录，获得7天免费试用
4. 输入激活码可以延长使用时间

---

## 两种使用方式

### 方式一：在线画图（推荐，最简单）

不需要安装任何额外软件，只需填一个密钥就能用：

1. 打开软件 → 点"设置"
2. 找到"云端画图"，打开开关
3. 选一个画图服务商（推荐"硅基流动"，有免费额度）
4. 填入API密钥（去对应网站注册就能拿到）
5. 搞定！可以直接用了

**密钥获取地址：**
- 硅基流动（推荐）：cloud.siliconflow.cn — 注册送免费额度
- DeepSeek写文案：platform.deepseek.com — 便宜好用
- 智谱GLM：open.bigmodel.cn — 有免费额度

### 方式二：本地画图（需要显卡，适合进阶用户）

需要自己安装：
- Stable Diffusion WebUI（画图软件）
- SD模型文件（画图用的素材库）
- Ollama + 模型（写文案用，可选）

安装后软件会自动检测，不需要额外配置。

## 遇到问题？

### 打不开软件
- 双击 `环境自检修复.bat`，会告诉你哪里有问题
- 杀毒软件拦截了？把整个文件夹加到信任列表
- 从网盘下载的文件被锁定了？双击 `首次运行引导.bat` 会自动解锁

### 画图失败
- 检查API密钥是否填对
- 如果用本地SD，确认SD WebUI已经启动
- 检查网络是否正常

### 登录失败
- 检查网络连接
- 确认用户名密码正确

### 软件卡住
- 关掉重新打开
- 还是不行？双击 `环境自检修复.bat` 检查一下
