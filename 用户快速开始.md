# 短视频生成器 - 使用说明

---

## 第一步：解压

1. 下载压缩包
2. **使用 7-Zip 或 WinRAR 解压**（不要用其他压缩软件的"快速解压"，会跳过文件校验）
   - 7-Zip 下载：https://7-zip.org （免费）
3. 建议解压到纯英文路径，比如 `D:\VideoGenerator\`
4. 不要解压到桌面或含中文的路径，可能出问题
5. **解压时请暂时关闭杀毒软件**（特别是 Windows Defender），否则可能误删关键文件
6. **解压完成后，双击 `检查环境.bat` 验证运行环境是否正常**
   - 如果提示文件缺失或损坏，请关闭杀毒后重新解压

## 第二步：启动

**双击 `启动.vbs`** 即可打开软件

> 首次打开？双击 **`!!!首次使用点我.bat`**，会自动帮你检查环境并引导操作

> **注意：首次使用需要联网**（注册账号和登录验证需要网络连接）

## 第三步：注册登录

1. 打开软件后会弹出登录窗口
2. 点 **"注册"** 标签页，填用户名、邮箱和密码
   - 密码要求：至少8位，必须包含大写字母、小写字母和数字
3. 注册完自动登录，获得 **7天免费试用**
4. 忘记密码？点登录页的 **"忘记密码?"** 通过邮箱验证码重置

- SD模型文件（画图用的素材库）
- Ollama + 模型（写文案用，可选）

安装后软件会自动检测，不需要额外配置。

## 制作视频的步骤

1. **导入音频** — 点左侧 **"📂 导入音频"**，选择你的音频文件
2. **等待分析** — 软件自动把音频转成文字，然后AI分析内容
3. **生成配图** — AI根据内容为每个画面生成图片
4. **合成视频** — 自动把图片、转场、音频合成为完整视频
5. **查看结果** — 点 **"📁 打开文件夹"** 查看生成的视频

> 💡 全程自动，你只需要导入音频然后点 **"🎞️ 生成视频"** 就行

## 遇到问题？

### 打不开软件
- 双击 `检查环境.bat`，会告诉你哪里有问题
- 杀毒软件拦截了？把整个文件夹加到**信任列表**
- 从网盘下载的文件被锁定了？右键压缩包 → 属性 → 勾选"解除锁定"后重新解压
- 提示文件缺失？可能是杀毒软件在解压时删除了文件，请关闭杀毒后重新解压
- 右键启动文件，选择 **"以管理员身份运行"** 试试

### 解压后文件不全
- 请使用 7-Zip 或 WinRAR 重新解压（不要用其他压缩软件的快速解压）
- 解压前先关闭杀毒软件的实时保护
- 解压后运行 `检查环境.bat` 验证环境
- 如果仍然有问题，请重新下载安装包

### 画图失败
- 检查API密钥是否填对（点"测试连接"确认）
- 如果用本地SD，确认SD WebUI已经启动且加了 `--api` 参数
- 检查网络是否正常

### 登录失败
- 检查网络连接
- 确认用户名和密码正确
- 忘记密码？点登录页的 **"忘记密码?"** 通过注册邮箱重置

### 软件卡住
- 关掉重新打开
- 还是不行？双击 `检查环境.bat` 检查一下

### 忘记密码
1. 在登录窗口点 **"忘记密码?"**
2. 输入你注册时的邮箱
3. 点"发送验证码"，去邮箱查收
4. 填入验证码和新密码，重置成功
