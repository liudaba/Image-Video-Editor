#### **⚠️ 需要额外安装的软件**(本地模式必需)

如果您想使用**纯本地模式**(不花钱),还需要安装以下两个软件:

##### **1. Ollama** (AI大语言模型)
**作用**: 分析音频内容,生成分镜脚本  
**下载地址**: https://ollama.com  
**安装步骤**:
```bash
# 1. 下载安装Ollama
# 2. 打开命令行,下载模型:
ollama pull qwen3:8b

# 推荐模型(选择一个):
# qwen3:8b    - 平衡性能和速度(推荐)
# gemma3:4b   - 轻量级,速度快
# llama3.1:8b - Meta官方模型
```

**大小**: 约4-8GB(取决于选择的模型)  
**硬件要求**: NVIDIA显卡(GTX 1060以上)

---

##### **2. Stable Diffusion WebUI** (图片生成)
**作用**: 根据提示词生成场景图片  
**安装教程**: 搜索"Stable Diffusion WebUI 安装教程"  
**关键步骤**:
```bash
# 启动时必须加 --api 参数
webui-user.bat --api
```

**大小**: 约10GB+(含Python环境和SD模型)  
**硬件要求**: NVIDIA显卡(RTX 3060以上推荐)

## 🎬 开始使用 - 3步生成视频

### 步骤1: 导入音频

1. 点击 **"📂 导入音频"** 按钮
2. 选择您的音频文件(MP3/WAV/M4A)
3. 等待加载完成

**支持的格式**:
- MP3 (最常用)
- WAV (无损音质)
- M4A (苹果设备)
- FLAC (高质量)

**文件大小限制**:
- 最大: 500MB
- 推荐: 5-50MB (3-10分钟音频)

---

### 步骤2: 调整设置(可选)

**基础设置**(新手可跳过):
- 视频分辨率: 1080p / 720p
- 艺术风格: 电影感/动漫/油画等13种
- 语速: 正常/快/慢

**高级设置**(点击"⚙️ 高级设置"):
- 云端模型配置
- 并发控制
- 缓存管理

---

### 步骤3: 开始生成

1. 点击 **"🚀 开始生成"** 按钮
2. 等待自动生成(10-20分钟)
3. 查看进度条和日志

**生成过程**:
```
🎵 第1步: 语音识别 (1-2分钟)
   ↓
🧠 第2步: AI分析内容 (1分钟)
   ↓
🎨 第3步: 生成分镜画面 (5-15分钟)
   ↓
🎬 第4步: 合成视频 (2-3分钟)
```

**完成后**:
```
✅ 视频生成完成!
📁 保存位置: output_project/
📹 文件名: video_20260503_184000.mp4
```

## ⚠️ 注意事项

### 🔴 必须遵守

1. **版权合规**
   - ❌ 不要使用有版权的音频
   - ✅ 使用原创或已授权的音频
   - ⚖️ 遵守当地法律法规

2. **系统要求**
   - 💻 Windows 10/11 (64位)
   - 💾 至少10GB可用磁盘空间
   - 🧠 建议8GB以上内存
   - 🎮 建议使用NVIDIA显卡(加速生成)

3. **网络要求**
   - 🌐 首次运行需要下载模型(2-5GB)
   - 🔄 后续使用可离线运行
   - ⚡ 建议稳定网络连接

---

### 🟡 建议使用

1. **音频质量**
   - ✅ 清晰的录音效果
   - ✅ 背景噪音少
   - ✅ 语速适中(每分钟150-200字)

2. **生成环境**
   - ✅ 关闭其他大型软件(释放内存)
   - ✅ 保持电源连接(笔记本)
   - ✅ 预留足够时间(10-20分钟)

3. **存储空间**
   - ✅ 每个视频约100-500MB
   - ✅ 定期清理旧视频
   - ✅ 备份重要作品

---

### 🟢 优化技巧

1. **提升生成速度**
   - 使用SSD硬盘
   - 关闭后台程序
   - 选择较低分辨率(720p)

2. **提升视频质量**
   - 使用高质量音频
   - 选择合适的艺术风格
   - 调整分镜密度

3. **节省时间**
   - 批量处理多个音频
   - 使用预设模板
   - 启用缓存功能
