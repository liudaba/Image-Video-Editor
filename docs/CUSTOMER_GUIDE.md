#### 3️⃣ **试用期检查**
```python
# 每次使用前检查
def check_license():
    if now > trial_end:
        return {
            "valid": False,
            "message": "试用期已结束,请购买正式版"
        }
    
    days_left = (trial_end - now).days
    return {
        "valid": True,
        "type": "trial",
        "days_left": days_left,
        "message": f"试用期剩余 {days_left} 天"
    }
```
