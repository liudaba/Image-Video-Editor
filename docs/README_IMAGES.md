## 📐 截图规范

### 尺寸要求
- **最小分辨率**: 1280x720
- **推荐分辨率**: 1920x1080
- **格式**: PNG (无损)
- **文件大小**: < 2MB/张

### 视觉规范
1. **窗口边框**: 保留Windows窗口边框,体现真实感
2. **背景**: 使用浅色桌面背景,避免干扰
3. **字体**: 确保所有文字清晰可读(不小于12px)
4. **高亮**: 可用红色框标注重点功能区域
5. **一致性**: 所有截图使用相同的主题和缩放比例

### 命名规范
```
docs/screenshots/
├── main_interface.png          # 主界面
├── advanced_settings.png       # 高级设置
├── generation_process.png      # 生成过程
├── cloud_model_config.png      # 云端模型配置(可选)
└── style_presets.png           # 风格预设展示(可选)
```

## 🎬 视频制作规范

### 技术参数
- **分辨率**: 1920x1080 (1080p)
- **帧率**: 30fps
- **编码**: H.264
- **格式**: MP4
- **时长**: 30-60秒/个
- **文件大小**: < 50MB/个

### 内容结构
```
0-5秒:   开场标题 + 案例简介
5-15秒:  展示输入音频(波形图或文件图标)
15-35秒: 快进展示生成过程(分镜→图片→视频)
35-50秒: 播放最终成品视频精彩片段
50-60秒: 结尾总结 + GitHub链接
```

### 字幕建议
- **字体**: 微软雅黑 /思源黑体
- **大小**: 24-32px
- **颜色**: 白色文字 + 黑色描边
- **位置**: 底部居中
- **时长**: 每句3-5秒

### 背景音乐
- **风格**: 轻快、科技感
- **音量**: 比主音频低20-30dB
- **推荐平台**: 
  - YouTube Audio Library (免费)
  - Epidemic Sound (付费,质量高)
  - 爱给网 (中文,部分免费)
