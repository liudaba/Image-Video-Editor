# 🎬 短视频生成器 - 快速上手指南

> 把一段音频变成带画面的短视频，AI全自动完成

---

## 三步出视频

```
1️⃣ 双击 start.bat 启动程序
2️⃣ 点击「📂 导入音频」选择音频文件
3️⃣ 点击「🎞️ 生成视频」等待完成
```

视频保存在 `output_project/` 文件夹，点击「📁 打开文件夹」查看。

---

## 使用前准备

| 必装软件 | 说明 |
|---------|------|
| Python 3.10+ | python.org 下载 |
| SD WebUI | 启动时加 `--api` 参数 |
| Ollama（本地模式） | 或使用云端大模型替代 |

> 💡 首次运行双击 `检查环境.bat` 检查依赖，缺包执行 `pip install -r requirements.txt`

## 两种AI模式

| | 本地模式 | 云端模式 |
|--|---------|---------|
| AI大脑 | Ollama（免费） | DeepSeek/GPT-4o等（付费） |
| 语音识别 | 本地Whisper | OpenAI Whisper API |
| AI画图 | 本地SD WebUI | 硅基流动/DALL-E等 |
| 费用 | 免费 | 按量付费（很低） |
| 要求 | 需要GPU | 只需联网 |

> 三个云端开关独立控制，可自由组合。全云端模式无需本地GPU。

## 高级设置要点

### 🎨 绘图设置
- **模型**：下拉菜单自动显示类型标签 `[SD1.5]` `[SDXL]` `[Flux]` `[SD3]`
- **分辨率**：默认1920×1080，显存不足可降至1280×720

### 🎭 风格设置
13种预设风格可叠加，选非写实风格（皮克斯/吉卜力等）时自动调整提示词

### 🎯 主题自定义（重要！）

| 设置项 | 怎么填 | 示例 |
|--------|--------|------|
| **核心主题** | 一句话概括视频核心论点 | `人类与病毒的微妙共存关系` |
| **视觉基调** | 2-3个情绪词 + 1-2个风格词 | `紧张压抑，冷色调，纪录片质感` |

> ⚡ 填写越具体，生成的图片越贴合内容。留空则AI自动分析。
>
> ❌ 不好：`科学` `好看` — 太模糊，AI不知道画什么
>
> ✅ 好的：`太空探索的勇气与代价` `温暖治愈，暖光，人文关怀`

### ⚡ 并发设置
- 分镜线程数：默认16，显存不足降到4-8
- 提示词线程数：默认4

### 📝 提示词设置
- **SD提示词**（默认）：自动适配SD1.5/SDXL/Flux/SD3
- **ARV写实提示词**：照片级真实感，选后风格选项禁用

## 云端大模型配置

1. 打开 ⚙️高级设置 → ☁️云端大模型区域
2. 选择服务商 → 输入API Key → 选模型
3. 点击 🔗测试连接 → 勾选启用
4. 点 ✅应用设置

**推荐**：DeepSeek（性价比最高）或 智谱AI GLM-4-Flash（免费）

| 服务商 | 注册地址 |
|--------|---------|
| DeepSeek | platform.deepseek.com |
| 智谱AI | open.bigmodel.cn |
| 硅基流动 | cloud.siliconflow.cn |

## 常见问题速查

| 问题 | 解决方法 |
|------|---------|
| Ollama未连接 | 启动Ollama，或切换云端模式 |
| 图片卡住不动 | 检查SD WebUI是否运行、API地址是否正确 |
| 图片和内容对不上 | 填写主题自定义的核心主题和视觉基调 |
| 显存不足 | 降低分辨率、减少线程数、启用云端 |
| 视频没声音 | 检查原始音频文件是否正常 |
| 重新开始 | 点🗑️清除数据，重新导入音频 |
