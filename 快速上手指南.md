# 🎬 短视频生成器 - 快速上手指南

> 把一段音频变成带画面的短视频，AI全自动完成

---

## 🚀 三步出视频

```
1️⃣ 双击「启动.vbs」打开软件
2️⃣ 点击「📂 导入音频」选择音频文件
3️⃣ 点击「🎞️ 生成视频」等待完成
```

✅ 视频保存在 `output_project/` 文件夹，点击「📁 打开文件夹」查看。

---

## 📦 首次使用

### 第一步：解压

1. 下载压缩包，**右键 → 属性 → 勾选"解除锁定"**（重要！）
2. 用 **7-Zip** 或 **WinRAR** 解压（不要用其他压缩软件的快速解压）
3. 解压到纯英文路径，比如 `D:\VideoGenerator\`
4. **解压时请暂时关闭杀毒软件**，否则可能误删关键文件

### 第二步：启动

- **双击 `启动.vbs`** 即可打开软件
- 首次使用？双击 **`!!!首次使用点我.bat`**，自动检查环境

### 第三步：注册登录

1. 打开软件后弹出登录窗口
2. 点 **"注册"** 标签页，填用户名、邮箱和密码
   - 密码要求：至少8位，必须包含大写字母、小写字母和数字
3. 注册完自动登录，获得 **15天免费试用**
4. 忘记密码？点登录页的 **"忘记密码?"** 通过邮箱验证码重置

> ⚠️ **首次使用需要联网**（注册和登录需要网络连接）

不需要安装任何额外软件，只需填几个密钥：

1. 打开软件 → 点 **"⚙️ 高级设置"**

## 🎞️ 制作视频的步骤

1. **导入音频** — 点左侧 **「📂 导入音频」**，选择你的音频文件
2. **等待分析** — 软件自动把音频转成文字，然后AI分析内容
3. **生成配图** — AI根据内容为每个画面生成图片
4. **合成视频** — 自动把图片、转场、音频合成为完整视频
5. **查看结果** — 点 **「📁 打开文件夹」** 查看生成的视频

> 💡 全程自动，你只需要导入音频然后点 **「🎞️ 生成视频」** 就行

## ❓ 遇到问题？

| 问题 | 解决方法 |
|------|---------|
| 打不开软件 | 双击 `检查环境.bat` 检查；关闭杀毒后重新解压；右键→以管理员身份运行 |
| 解压后文件不全 | 用7-Zip重新解压；解压前关闭杀毒；先右键压缩包→属性→勾选"解除锁定" |
| 画图失败 | 检查API密钥是否正确；用本地SD需确认SD WebUI已启动且加了 `--api` 参数 |
| 登录失败 | 检查网络连接；确认用户名密码正确；忘记密码点「忘记密码?」 |
| 软件卡住 | 关掉重新打开；双击 `检查环境.bat` 检查 |
| 如何购买会员 | 软件内点「⭐ 购买会员」；或联系客服获取激活码 |
