## 📦 使用前准备

### **方式一：云端模式（推荐新手）**

**优势：**
- ✅ 无需安装任何软件
- ✅ 对显卡无要求
- ✅ 5分钟快速上手

**操作步骤：**
1. 启动程序后，点击「⚙️ 高级设置」
2. 勾选「☁️ 启用云端大模型」和「☁️ 启用云端生图」
3. 填写 API Key（推荐 DeepSeek 或 智谱AI）
4. 点击「🔗 测试连接」确认成功
5. 开始使用！

> 💡 **API Key 获取指南**：
> - **DeepSeek**（性价比高）：https://platform.deepseek.com
> - **智谱AI GLM-4-Flash**（免费额度）：https://open.bigmodel.cn
> - **硅基流动**（多模型支持）：https://cloud.siliconflow.cn

#### 2️⃣ SD WebUI（AI画笔）

**安装步骤：**

**方法一：官方原版**
```bash
# 1. 克隆仓库
git clone https://github.com/AUTOMATIC1111/stable-diffusion-webui.git

# 2. 进入目录
cd stable-diffusion-webui

# 3. 编辑启动脚本（关键步骤！）
# 用记事本打开 webui-user.bat 文件
# 找到这一行：set COMMANDLINE_ARGS=
# 修改为：set COMMANDLINE_ARGS=--api
# 保存文件

# 4. 双击运行 webui-user.bat
```

**💡 如何添加 --api 参数（详细教程）：**

1. **找到文件**：在 SD WebUI 文件夹中找到 `webui-user.bat`
2. **右键编辑**：右键点击 → 选择"编辑"或"用记事本打开"
3. **查找参数**：按 `Ctrl+F` 搜索 `COMMANDLINE_ARGS`
4. **添加参数**：在等号后面加上 `--api`
   ```batch
   # 修改前
   set COMMANDLINE_ARGS=
   
   # 修改后
   set COMMANDLINE_ARGS=--api
   ```
5. **保存关闭**：按 `Ctrl+S` 保存，然后关闭记事本
6. **启动程序**：双击 `webui-user.bat` 运行

> ⚠️ **重要提示**：如果不加 `--api` 参数，本程序无法连接到 SD WebUI！

## 🤖 两种AI模式对比

| 特性 | 本地模式 | 云端模式 |
|------|---------|---------|
| **AI大脑** | Ollama（免费） | DeepSeek/GPT-4o等（付费） |
| **语音识别** | 本地Whisper | OpenAI Whisper API |
| **AI画图** | 本地SD WebUI | 硅基流动/DALL-E等 |
| **费用** | 完全免费 | 按量付费（很低） |
| **硬件要求** | 需要GPU（至少4GB显存） | 只需联网 |
| **隐私性** | 数据完全本地 | 需上传到云端 |
| **速度** | 取决于显卡性能 | 通常更快 |

> 💡 **混合模式**：三个云端开关独立控制，可自由组合。例如：云端LLM + 本地SD，兼顾速度与成本。

## ⚙️ 高级设置要点

### 🎨 绘图设置
- **模型**：下拉菜单自动显示类型标签 `[SD1.5]` `[SDXL]` `[Flux]` `[SD3]`
- **分辨率**：默认1024×576，显存不足可降至更小，范围256-4096

### 🎭 风格设置
13种预设风格可叠加，选非写实风格（皮克斯/吉卜力等）时自动调整提示词

### 🎯 主题自定义（重要！提升准确度）

| 设置项 | 怎么填 | 示例 |
|--------|--------|------|
| **核心主题** | 一句话概括视频核心论点 | `人工智能如何改变教育方式` |
| **视觉基调** | 2-3个情绪词 + 1-2个风格词 | `温暖治愈，暖光，人文关怀` |

> ⚡ 填写越具体，生成的图片越贴合内容。留空则AI自动分析。
>
> ❌ **不好**：`科学` `好看` — 太模糊，AI不知道画什么
>
> ✅ **好**：`科技与自然的和谐共生` `清新明亮，绿色调，纪录片质感`

### ⚡ 并发设置
- 分镜线程数：默认8，显存不足降到4-6

### 🎙️ 语音模型选择
| 模型 | 磁盘大小 | GPU显存占用 | 中文识别 | 适用显卡 |
|------|----------|-------------|----------|----------|
| **small** | ~460 MB | ~1.5 GB | 较好 | 4GB+ 显存 |
| **medium**（默认） | ~1.4 GB | ~4.3 GB | 优秀 | 8GB+ 显存 |
| **large** | ~2.9 GB | ~9 GB | 最优 | 12GB+ 显存 |

> ⚠️ 显存不足时模型会自动回退CPU模式（速度较慢），建议根据显卡显存选择合适模型。

### 📝 提示词设置
- **SD提示词**（默认）：自动适配SD1.5/SDXL/Flux/SD3
- **ARV写实提示词**：照片级真实感，选后风格选项禁用

## ❓ 常见问题速查

### Q1：程序提示"Ollama未连接"？

**解决方法（二选一）：**
- **方案A（推荐）**：在高级设置中启用云端模式，无需安装Ollama
- **方案B**：安装并启动Ollama（访问 ollama.com 下载）

> 💡 程序会自动尝试启动Ollama，如果失败会弹窗提醒

---

### Q2：生成图片时卡住不动？

**检查清单：**
- ✅ 是否启用了"云端生图"？（推荐新手启用）
- ✅ 如果使用本地SD，确认SD WebUI已启动且加了 `--api` 参数
- ✅ API地址是否正确（默认 `http://127.0.0.1:7860`）

---

### Q3：生成的图片和音频内容对不上？

**优化方法：**
1. 在"高级设置" → "主题自定义"中，手动填写核心主题（越具体越好）
   - ❌ 不好：`科学`
   - ✅ 好：`人类与病毒的共存关系`
2. 尝试切换更强的云端模型（如 GPT-4o）

---

### Q4：视频合成失败？

**检查：**
- ✅ 原始音频文件是否还在（不要删除）
- ✅ 磁盘空间是否充足（至少留5GB）
- ✅ 图片是否全部生成完成（看日志有没有报错）

**解决方法**：点击垃圾桶🗑️清除数据，重新导入音频从头开始

---

### Q5：如何重新开始？

点击音频旁边的 **🗑️ 垃圾桶按钮** ，清除当前数据后重新导入音频即可

> 💡 旧文件会移到"垃圾桶"文件夹，可以按时间找回

---

### Q6：试用期结束后怎么办？

试用期为7天或生成50个视频（先到者为准）。到期后会弹出购买窗口，可选择：
- 月付套餐
- 年付套餐（更优惠）

购买后输入授权码即可继续使用。

| 模式 | 全流程耗时 |
|------|-----------|
