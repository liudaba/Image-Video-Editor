# 使用说明书

> 把音频变成带画面的短视频，AI自动完成：转文字 → 配图 → 合成视频

---

## 两种运行模式

| | 云端模式（推荐新手） | 本地模式（需显卡） |
|---|---|---|
| 需要安装 | 什么都不用装 | SD WebUI + Ollama |
| 费用 | 几毛钱一条视频 | 免费 |
| 适合 | 新手、无显卡用户 | 有显卡的进阶用户 |

> 三个功能（写文案、画图、语音识别）各有独立开关，可以混搭。比如：云端写文案 + 本地画图。

## 一、安装与启动

### 1.1 解压

1. 右键压缩包 → **属性** → 勾选**「解除锁定」** → 确定（不做可能无法运行！）
2. 用 **7-Zip** 或 **WinRAR** 解压到**英文路径**，如 `D:\VideoGenerator\`
3. 解压前**先关杀毒软件**，否则可能误删文件
4. 解压后可运行 `CheckEnv.bat` 检查环境

### 1.2 启动

| 场景 | 操作 |
|------|------|
| 日常使用 | 双击 **`start.vbs`** |
| 首次使用 | 双击 **`FirstRunSetup.bat`**（7步引导） |
| 排查问题 | 双击 **`start.bat`**（带错误提示） |

### 1.3 注册

- 打开软件 → 注册 → 填用户名、邮箱、密码（至少8位）
- 注册后自动获得15天免费试用
- 忘记密码？点登录页「忘记密码?」

## 二、云端模式：填写 API Key

云端模式不用装任何东西，只需填入 API Key。

### 2.1 API Key 是什么？

一串密钥，用来调用云端AI服务。去服务商网站注册就能免费获取，按使用量计费，一条视频几毛钱。

### 2.2 怎么获取？

1. 去服务商网站注册账号
2. 进入「控制台」→「API Key管理」→「创建Key」
3. 复制 Key 保存好（关了页面就看不到了）

### 2.3 怎么填入软件？

1. 打开软件 → 点「高级设置」
2. **写文案**：勾选「启用云端LLM」→ 选服务商 → 粘贴Key → 选模型
3. **画图**：勾选「启用云端生图」→ 选服务商 → 粘贴Key → 选模型
4. **语音识别**（可选）：勾选「启用云端语音识别」→ 粘贴Key
5. 点「测试连接」确认能用 → 点「应用设置」

> Key 在本地加密保存，不会上传。

## 三、本地模式：安装软件

### 3.1 画图 — SD WebUI（必需，需显卡）

**秋叶整合包（推荐）：**
1. 搜"秋叶SD启动器"下载
2. 启动器 → 高级选项 → 勾选"开启API" → 一键启动

**官方原版：**
1. 下载 stable-diffusion-webui
2. 改 `webui-user.bat` 里 `COMMANDLINE_ARGS=--api`
3. 双击运行

> 必须加 `--api` 参数，否则软件无法调用。

### 3.2 写文案 — Ollama（可选）

1. 去 ollama.com 下载安装
2. 命令行执行 `ollama pull gemma3:4b`
3. 软件会自动检测

> 不装Ollama也可以，启用云端LLM替代即可。

## 四、制作视频

1. **导入音频** — 点「导入音频」，选音频文件（mp3、wav等）
2. **自动转文字** — 内置Whisper，无需配置
3. **AI写文案** — 自动生成画图提示词
4. **AI画图** — 自动为每句话配图
5. **合成视频** — 图片+转场+音频自动合成
6. **查看结果** — 点「打开文件夹」

> 全程自动，导入音频 → 点「生成视频」就行

## 五、高级设置

### 5.1 绘图

- **模型**：下拉菜单有类型标签 `[SD1.5]` `[SDXL]` `[Flux]`
- **分辨率**：默认1024x576，显存不够可降低
- **提示词类型**：SD提示词（默认）或 ARV写实提示词（照片级）

### 5.2 风格

13种预设风格可叠加

### 5.3 主题自定义（提升准确度）

| 设置项 | 怎么填 | 示例 |
|--------|--------|------|
| 核心主题 | 一句话概括视频主题 | `人工智能如何改变教育` |
| 视觉基调 | 情绪词+风格词 | `温暖治愈，暖光，人文关怀` |

> 越具体越好。`科学`太模糊，`科技与自然的和谐共生`就很好。

### 5.4 语音模型

| 模型 | 磁盘 | 显存 | 中文识别 | 适合显卡 |
|------|------|------|----------|---------|
| small | 460MB | 1.5GB | 较好 | 4GB+ |
| medium（默认） | 1.4GB | 4.3GB | 优秀 | 8GB+ |
| large | 2.9GB | 9GB | 最优 | 12GB+ |

> 显存不够会自动用CPU（较慢）

## 七、常见问题

**打不开软件**
→ 运行 `CheckEnv.bat`；加杀毒信任；右键以管理员运行

**API Key 相关**
→ 去服务商网站注册获取 → 软件内「高级设置」粘贴 → 点「测试连接」确认
→ Key加密保存在本地，不会上传
→ 还是报错？检查Key对不对、网络通不通、账户有没有余额

**画图失败**
→ 检查Key对不对（点测试连接）；本地SD要加 `--api` 参数

**登录失败**
→ 检查网络；忘记密码点「忘记密码?」

**图片和内容对不上**
→ 填写「核心主题」，越具体越好

**视频合成失败**
→ 确认音频文件还在；磁盘至少留5GB；点垃圾桶清除重来

**想重新开始**
→ 点音频旁的垃圾桶按钮清除；旧文件在 trash/ 里可恢复
→ 点控制台标题栏🗑️按钮打开垃圾桶浏览器
